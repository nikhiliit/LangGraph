# LangGraph Research Projects

This repository contains two LangGraph-based research paper analysis agents with different architectures and capabilities.

## Agentic_Research_Evaluator
Enhanced research assistant with hallucination prevention using an evaluator node that validates responses against source PDF content.
Provides iterative refinement and ensures all answers are grounded in the provided documents with structured feedback.

## Agentic_Research_Exploration
Clean web interface research paper analysis agent with natural language Q&A capabilities, PDF processing, and conversation memory.
Features professional Gradio UI with user registration, LaTeX rendering, and SQLite checkpointing for persistent conversations.
